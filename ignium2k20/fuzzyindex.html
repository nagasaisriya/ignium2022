Fuzzy Code



Is coding your cup of tea? Do you have bytes of C for breakfast? If you do, then join a bunch of other nerdy coders to bring out the best in you. IGNIUM 2K18 brings Fuzzy Code, where you get to challenge your coding, innovation and management abilities, all at once.

Rules



ELIGIBILITY:

The event is open to all regular B.Tech students who are acquainted with coding in C language.

EVENT DESCRIPTION:
Round 1: Team is provided with an erroneous code and team has to debug the code in the given stipulated time.
Round 2: One of the team members is provided with a problem statement, which is unknown to the other participant. That team member has to code for half of the given time and the next team member has to finish the solution to the statement given in the next half of the time.
